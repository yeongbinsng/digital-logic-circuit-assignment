# 디지털논리회로 과제물 1

1. 다음과 같은 문제의 유형과 동일한 새로운 문제를 출제하고 해당 문제에 대한 답을 제출하는 것이 과제입니다.
2. 출제된 문제는 지금까지 어느 곳에서도 출제가 되지 않은 문제이어야 하며, 이것을 생성형 AI를 통해 증명하여야 합니다. 새로은 문제를 생성형 AI를 통해 만들 수 있으나, 그 경우, 반드시 다른 생성형 AI로 검증해야 합니다. 예를들어, ChatGPT에서 문제를 만들면, 새로운 문제라는 검증은 Gemini로 해야 합니다.
3. 출제된 문제와 정답은 제시된 공유게시판에 올려야 과제물 제출이 완료됩니다.
4. 새로운 문제라는 증빙파일(생성형 AI 응답에 대한 캡쳐본)은 조교에게 이메일로 제출하시기 바랍니다.
